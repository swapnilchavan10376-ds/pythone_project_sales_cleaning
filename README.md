Project Description: Advanced Sales Data Analysis Dashboard

This project analyzes an online retail sales dataset using Python, Pandas, NumPy, Matplotlib, and Seaborn to generate business insights and reports. It demonstrates data cleaning, exploratory data analysis (EDA), statistical analysis, visualization, and report generation.

Objective

The main goal is to transform raw sales data into meaningful information that helps businesses understand:

Sales performance
Customer behavior
Product popularity
Country-wise sales trends
Monthly sales patterns
